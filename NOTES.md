## JavaScript Concepts

### Template Literals

- Use backticks: `string here`
- Interpolate with ${expression}
- Same as Python f-strings or C# $""
- Pure JavaScript feature (ES6)

Example:

```js
const name = 'Alice';
const greeting = `Hello, ${name}!`;
```
